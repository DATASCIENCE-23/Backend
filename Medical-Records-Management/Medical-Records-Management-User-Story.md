# Medical Records Management – User Stories

## Overview
The Medical Records Management module enables secure creation, storage, and access of patient medical data.  
It supports doctors, patients, and administrators while ensuring privacy, accuracy, and controlled access.

---

## Actors
- Doctor
- Patient
- Admin
- System

---

## Doctor User Stories

**US-01**  
As a doctor, I want to log in securely so that only authorized doctors can access patient medical records.

**US-02**  
As a doctor, I want to search for patients using ID or name so that I can quickly access their medical history.

**US-03**  
As a doctor, I want to create new medical records so that patient diagnoses and treatments are documented.

**US-04**  
As a doctor, I want to update existing medical records so that patient information remains accurate and up to date.

**US-05**  
As a doctor, I want to view a patient’s complete medical history so that I can make informed clinical decisions.

---

## Patient User Stories

**US-06**  
As a patient, I want to register in the system so that my medical records can be stored digitally.

**US-07**  
As a patient, I want to log in securely so that my personal health information is protected.

**US-08**  
As a patient, I want to view my medical records so that I can track my health history.

**US-09**  
As a patient, I want to view prescriptions and reports so that I can follow the doctor’s instructions correctly.

---

## Admin User Stories

**US-10**  
As an admin, I want to manage users so that only valid doctors and patients can access the system.

**US-11**  
As an admin, I want to assign and manage user roles so that access permissions are enforced correctly.

**US-12**  
As an admin, I want to monitor system activity so that misuse of medical data can be detected.

---

## System & Security User Stories

**US-13**  
As a system, I want to encrypt user passwords so that authentication data remains secure.

**US-14**  
As a system, I want to enforce role-based access control so that patients cannot modify medical records.

---

## Summary
The Medical Records Management module ensures secure handling of patient data, enables doctors to manage clinical information efficiently, allows patients to view their records transparently, and provides administrators with full control over access and security.
